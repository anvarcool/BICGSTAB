
Этот проект реализует параллельный алгоритм BiCGStab (Bi-Conjugate Gradient Stabilized) с использованием MPI для решения систем линейных алгебраических уравнений.
Метод использует два последовательных ортогональных направления и стабилизационный шаг для улучшения сходимости.

## Сборка

Пример сборки под Windows (Microsoft MPI):
```bash
g++ -I"C:\Program Files (x86)\Microsoft SDKs\MPI\Include" main.cpp algebra.cpp generate_system.cpp ^
    -L"C:\Program Files (x86)\Microsoft SDKs\MPI\Lib\x64" -lmsmpi -o main.exe
```

## Запуск

### 1. Случайная система (по умолчанию)
Генерация случайной матрицы **A** и вектора **b**.
```bash
mpiexec -n 12 ./main.exe
```

### 2. Считывание системы из файлов (-f)
Файлы **A.txt** и **b.txt** должны быть в текущей директории.

```bash
mpiexec -n 12 ./main.exe -f
```

### 3. Считывание системы с консоли (-c)
Ввод матрицы и вектора вручную.
```bash
mpiexec -n 12 ./main.exe -c
```
